# Satia Auto Wireframe

Clickable HTML wireframe for **Satia Auto** ERP / CRM / Fleet Management system.

## Included Pages

- Dashboard
- Vehicle Definition
- Parts Definition
- Organization Chart
- Employees
- Customers
- Leasing Companies
- Contracts & Sales
- Services

## Goal

This repository is prepared as a presentable prototype to demonstrate project progress in client meetings.

## How to Run

Open `index.html` in a browser.
